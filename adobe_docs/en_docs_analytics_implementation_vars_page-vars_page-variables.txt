Page variables determine the values for dimensions and metrics in reporting.
The following list are variables frequently used in implementations:
pageName
: The name of the page.
campaign
: Set this variable to a query string parameter for campaign tracking.
events
: Populate metrics for use in reporting.
products
: If you have an eCommerce site, set this variable when a visitor views or purchases a product.
